hexo安装教程

发表于 2017-10-21 | 分类于 hexo
  1. 安装node.js，一路安装即可，在命令行输入node验证是否安装成功；
  2. 安装git，一路安装即可，安装完配置环境变量，在path中添加git所在的bin目录（D:\Program Files\Git\bin），打开命令行输入git验证安装；
  3. 申请github账号，在其中添加公钥，使用git bash命令行生成公钥，输入ssh-keygen命令，一路回车将公钥报存在默认路径，输入命令cat ~/.ssh/id_rsa.pub查看公钥，将看到的一长串字符串输入到github设置中；
  4. 在github上创建一个空的repository，命名必须是github账户名.github.io，然后使用gitclone到本地已创建好的空文件夹hexo下；
  5. 安装hexo，打开git bash命令行切换到已创建好的空文件夹hexo下，依次输入命令npm install -g hexo，npm install hexo-deployer-git –save下载时间可能较长，安装好后输入hexo -v 验证安装；
  6. 切换到clone的github账户名.github.io文件夹下，输入命令hexo init；
  7. 配置_config.yml文件： 
    deploy:  
  type: git  
  repo: github上项目的git clone路径（git@github.com:luhu888/luhu888.github.io.git）      
branch: master
  8. git上传文件部署项目，依次输入命令hexo clean，hexo generate，hexo deploy；
  9. 部署成功，输入github账户名.github.io即可访问。
  10. 常用命令：hexo new filename（新增博文）
