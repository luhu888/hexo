Ubuntu14.04下安装git及同步代码

发表于 2017-10-22 | 分类于 git

Linux下安装git，并同步代码流程

1. sudo pip install git 安装git

2. ssh-key -C ‘你的邮箱地址’ -t rsa 创建验证的公钥

3. 之后在输入完这个命令以后会有Enter file in which to save the key（/home/haohao/.ssh/id_rsa)这个输入选项，并不用在意，这个是选择是否在默认地址创建文件还是在新输入的地方创建（新的文件名称），回车就好，就可以在默认的地址创建新的ssh文件

4. cd ~/.ssh

5. gedit id_rsa.pub, 在代码仓库创建新的ssh

6. 首先在想要的位置创建一个空文件夹，使用命令sudo git clone https://github.com/luhu888/SchoolMS /school 后面代表指定的空文件夹路径，或者开启命令行先进入指定的路径再进行clone操作，

7. 若没有创建公钥，或创建失败，直接使用该命令亦可，只不过clone的同时需要在命令行中输入用户名和密码。

8. 在项目.git的同级目录（即项目文件所在的根目录）下输入命令git pull 即可同步代码，注意clone的代码不能改动，否则只能删除重新clone，不能拉取更新代码。
